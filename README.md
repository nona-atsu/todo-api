# TODO-API
## Description
リストの作成、更新、削除、取得ができるAPIです。

## localでの起動方法
### Directory
```
pwd -> Users/XXXXXX/todo-api
```

### 起動
```
npm run dev
```

### curl
```
curl -s -X GET http://localhost:3000
```

## mysql&redis
### Directory

```
pwd -> /Users/XXXXXXXX/todo-api/docker/mysql
```
